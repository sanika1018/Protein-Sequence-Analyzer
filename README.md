# 🧬 Protein Sequence Analyzer CLI

A Linux-based bioinformatics command-line tool for analyzing protein sequences from FASTA files.

This project demonstrates core bioinformatics concepts using Python and Bash, including sequence parsing, amino acid analysis, and protein property calculations.

---

# 🚀 Features

- 📁 Interactive FASTA file input (file name or full path)
- 🧬 FASTA format validation
- 📊 Amino acid composition analysis
- ⚖️ Molecular weight calculation
- 💧 Hydrophobicity score computation
- ⚡ Charged residue ratio analysis
- 🔍 Motif detection (N-glycosylation sites)
- 📄 Automatic report generation in output folder
